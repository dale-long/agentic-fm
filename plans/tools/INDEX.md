# FileMaker Server Tools

Reference documents for the CLI tools available on FileMaker Server. These tools run **server-side** — the agent triggers them via OData → FM scripts → MBS Shell. Read individual files only when working on tasks that involve these tools.

| Tool | File | Purpose |
|---|---|---|
| **FMDeveloperTool** | [FMDeveloperTool.md](FMDeveloperTool.md) | File operations: saveAsXML, clone, copy, encrypt, recover, consistency check, size analysis, upload to FMS |
| **FMDataMigration** | [FMDataMigration.md](FMDataMigration.md) | Record-level data migration between source and clone files (schema deployment) |
| **FMSConfigUpgrader** | [FMSConfigUpgrader.md](FMSConfigUpgrader.md) | Upgrades FMS configuration files between server versions |
| **fmsadmin** | [fmsadmin.md](fmsadmin.md) | Server administration: host/close files, manage clients, backups, schedules, services |
